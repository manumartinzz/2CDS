Sobre o Curso

O curso técnico em Desenvolvimento de Sistemas forma profissionais para planejar, desenvolver e manter sistemas computacionais. Você aprenderá linguagens de programação, banco de dados, lógica e estrutura de software.

O que você vai aprender

  • Programação Web (HTML, CSS, JavaScript)
  • Desenvolvimento de aplicativos móveis
  • Banco de Dados e SQL
  • Projetos de software e lógica de programação

Áreas de Atuação

Empresas de tecnologia, startups, escritórios de TI e desenvolvimento de sistemas para organizações públicas e privadas.
